# Digital Portfolio

This is my digital portfolio website showcasing my skills, projects, and contact information.

## Sections

- **About**: Introduction and background.
- **Skills**: List of technical skills.
- **Projects**: Showcase of my work.
- **Contact**: How to reach me.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to View

Open `index.html` in your web browser.

## Customization

Replace placeholder text with your actual information. Add images to an `images` folder and link them in the HTML.
